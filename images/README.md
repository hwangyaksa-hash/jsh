# 이미지 가이드

## 필요한 이미지 파일

이 폴더에 다음 이미지들을 추가해주세요:

### 1. **hero-bg.jpg** (Hero 섹션 배경)
- **용도**: 홈페이지 최상단 히어로 섹션의 배경 이미지
- **권장 크기**: 1920x1080px 이상
- **설명**: 대표님의 사진이나 회사를 대표하는 이미지
- **효과**: 어두운 오버레이가 적용되어 20% 투명도로 배경에 표시됩니다

### 2. **ceo-profile.jpg** (대표 프로필)
- **용도**: 대표 소개 섹션의 프로필 이미지
- **권장 크기**: 600x800px (세로형 추천)
- **설명**: 대표님의 공식 프로필 사진 (정장 또는 비즈니스 캐주얼)
- **효과**: 모서리가 둥글게 처리되고 그림자 효과가 적용됩니다

### 3. **company-main.jpg** (회사 소개 이미지)
- **용도**: About 섹션의 메인 이미지
- **권장 크기**: 700x700px (정사각형 추천)
- **설명**: 회사 로고, 사무실, 팀 사진, 또는 대표적인 작업 장면
- **효과**: 둥근 모서리와 그림자 효과

## 이미지 추가 방법

1. 위의 파일명으로 이미지를 이 폴더(`images/`)에 저장
2. JPG 또는 PNG 형식 사용 가능
3. 파일명을 정확하게 맞춰주세요 (대소문자 구분)

## 추가 커스터마이징

### 배경 이미지 투명도 조절
`index.html` 파일에서 다음 부분을 찾아서 `opacity` 값을 조절하세요:
```css
.hero::before {
    opacity: 0.2; /* 0.1 (매우 투명) ~ 0.5 (덜 투명) */
}
```

### 이미지 크기 조절
각 섹션의 CSS에서 `max-width` 값을 조절하면 이미지 크기를 변경할 수 있습니다.

## 이미지가 없을 때

이미지를 아직 준비하지 못했다면:
1. 임시로 해당 이미지 경로를 주석 처리하거나
2. 플레이스홀더 이미지 서비스를 사용할 수 있습니다
   - 예: `https://via.placeholder.com/600x800?text=CEO+Profile`

## 이미지 최적화 팁

- 웹용으로 이미지를 최적화하여 파일 크기를 줄이세요
- JPG: 사진 이미지에 적합 (80-90% 품질 권장)
- PNG: 로고나 투명 배경이 필요한 경우 사용
- 온라인 도구: TinyPNG, Squoosh 등 활용
